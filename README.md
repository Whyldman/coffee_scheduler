# coffee_scheduler
Code to provision and set up our Belkin coffee maker following the end of Wemo

## provision coffee
Connect to the coffee maker's broadcasted WiFi
Run the `provision_coffee` script to connect the coffee maker to the home network

## schedule_coffee
Need to be conncted to the same 2.4 Ghz network as the coffee maker first.
Run it on a PC to schedule the brew. It saves to the coffee maker

## brew_now
Use the HTTP Shortcup App on Android to create a "Brew Now" button
